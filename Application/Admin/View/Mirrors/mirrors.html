镜像配置教程

在左边的“repo文件下载”栏目上，根据需要下载相应的repo文件，然后使用xshell，配合xftp， 将对应的repo文件上传到Linux的/etc/yum.repos.d/目录下，最后使用命令：
yum clean all
yum makecache
更新yum源缓存，没报错则证明yum源配置成功。
提示：将repo文件上传到Linux目录时请先清空原有的repo（慎删！），命令：
rm -rf * /etc/yum.repos.d/
